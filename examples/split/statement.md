**Statement**  

Given an array, find the longest subsequence consisting of distinct elements. If there are multiple, any one will be accepted.

**Input Format**  

    T
    N
    A1 A2 ... AN
    N
    A1 A2 ... AN
    ...
    N
    A1 A2 ... AN

**Output Format**  

    M
    B1 B2 ... BM
    M
    B1 B2 ... BM
    ...
    M
    B1 B2 ... BM


**Constraints**  

$1 \le T \le 10^5$  
$1 \le N \le 10^5$  
$\sum N \le 5\cdot 10^5$  
$-10**9 \le A_i \le 10^9$  

**Subtask 1**: $N \le 10$  
**Subtask 2**: $N \le 1000$  
**Subtask 3**: No additional constraints
